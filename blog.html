What is DOM and what is the purpose of DOM?

Web documents include a programming interface called the Document Object Model (DOM). Since that software may alter the document's structure, appearance, and content, it serves as a representation of the page. Because the document is represented by the DOM as nodes and objects, computer languages may communicate with the page.

What is event bubble?

The DOM has a notion known as event bubbling (Document Object Model). It occurs when an element receives an event, and that event propagates up the DOM tree through the element's parent and ancestor elements until it reaches the root element.

What is a callback function and why will you use it?

A callback function is a function that receives an input from another function and is subsequently called from inside the outer function to finish a task or activity. The aforementioned illustration is a synchronous callback since it is executed right away.

Due to the fact that many JavaScript activities are asynchronous, callback methods are required. Instead, it runs in the background when the rest of the code is being executed. The function of a callback is to run code in response to an event.
